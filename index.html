Rusty Lake Website Project Portfolio

Betty Fang | Web Development Project | 2026

This project was created to design and develop an immersive themed website based on the Rusty Lake game universe. The goal was to explore visual storytelling through web design and apply HTML and CSS techniques to build a professional-looking site.

Rusty Lake is known for its dark atmosphere, symbolism, and psychological themes. I used a dark red and black color scheme, glow effects, and subtle animations to recreate the mysterious feeling of the game series. Created structure using HTML. Styled layout using CSS. Added animations with keyframes. Tested locally using VS Code. Published using GitHub Pages. Through this project, I improved my understanding of webpage structure, CSS styling techniques, and design thinking. I also learned how to translate a narrative theme into visual presentation. In the future, I would like to improve responsiveness and add JavaScript interactivity.
