Project 1 — Rsty Lake Game Website

Project Overview

This project is an immersive themed website inspired by the surreal and psychological style of Rusty Lake. The focus was to create a dark visual atmosphere through color design, layout composition, and subtle motion effects.

The site emphasizes immersive storytelling and artistic interface design, aiming to translate a game world's emotional tone into a web experience.

Design Focus

• Dark atmospheric theme
• Red-highlight visual identity
• Minimalist layout structure
• Mood-driven visual storytelling
