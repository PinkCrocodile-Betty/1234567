Portfolio Introduction

This portfolio presents my personal web development project inspired by the mysterious and surreal Rusty Lake game series. The goal was to create an immersive experience using only HTML and CSS while maintaining a consistent dark and red-themed visual style.

The website emphasizes typography, layout, and minimalistic animation to convey the game's psychological and mysterious atmosphere. Through this project, I explored visual storytelling, interactive design, and website structure.

Project

You can view the live Rusty Lake project website here: Rusty Lake Website.

The project features:

  • Red-on-black dark theme consistent with the Rusty Lake universe
  • Structured sections including Home, Story, and Games
  • Subtle glow and hover effects for text and links
