Rusty Lake Website Project Portfolio

Ningxin Fang | Web Development Project | 2026

Project Overview

This project was created to design and develop an immersive themed website based on the Rusty Lake game universe. The goal was to explore visual storytelling through web design and apply HTML and CSS techniques to build a professional-looking site.

Design Inspiration

Rusty Lake is known for its dark atmosphere, symbolism, and psychological themes. I used a dark red and black color scheme, glow effects, and subtle animations to recreate the mysterious feeling of the game series.

Development Process

  • Created structure using HTML
  • Styled layout using CSS
  • Added animations with keyframes
  • Tested locally using VS Code
  • Published using GitHub Pages

Technical Features

  • Responsive grid layout
  • Glow animation effects
  • Fixed navigation bar
  • Dark immersive theme design

Reflection

Through this project, I improved my understanding of webpage structure, CSS styling techniques, and design thinking. I also learned how to translate a narrative theme into visual presentation. In the future, I would like to improve responsiveness and add JavaScript interactivity.
