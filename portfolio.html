Web Development Portfolio

Name: Betty Fang

Project Title: Rusty Lake Universe Website

Project Overview

This project is a personal website inspired by the game series Rusty Lake. The website introduces the background story and major titles of the series.

Purpose

The purpose of this project was to practice basic web development skills using HTML and CSS, and to learn how to structure a complete website independently.

Development Process

I designed the layout structure first, then built the webpage using HTML. After that, I applied CSS styling to improve readability and layout. The final version was uploaded to GitHub and deployed using GitHub Pages.

Skills Used

  • HTML structure
  • Basic CSS styling
  • Website layout design
  • GitHub Pages deployment

Challenges

One challenge was understanding how to deploy the website correctly using GitHub Pages. I also learned how different sections of a webpage work together to create a complete structure.

Reflection

Through this project, I gained confidence in building websites independently. I improved my understanding of webpage structure and learned how to connect design ideas with technical implementation.
