Introduction

This portfolio presents my web design project inspired by the mysterious atmosphere of Rusty Lake. I created this website to explore immersive visual storytelling and atmospheric interface design.

The project was developed using HTML and CSS in Visual Studio Code. I focused on dark visual tone, red highlights, smooth layout structure, and subtle animation effects to match the psychological style of the series.
